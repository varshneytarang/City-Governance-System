## **INTERACTION TIPS FOR VIDEO**

### **Smooth Navigation:**
1. **Slow Scroll:** Scroll slowly (1-2 seconds per section)
2. **Pause on Key Elements:** Hover 2-3 seconds on important items
3. **Smooth Transitions:** Use browser back/forward or nav menu, not URL bar
4. **Cursor Visibility:** Keep cursor movements deliberate and visible

### **Hover Effects to Highlight:**
- Department cards → Show tooltip/description
- Stats → Show trend details
- Agent constellation → Show agent details
- Quick action buttons → Show expanded info
- Activity items → Show full details

### **Timing Guide:**
- Landing page: 3-4 minutes
- Login: 20 seconds
- Dashboard: 2-3 minutes
- First department (detailed): 1.5 minutes
- Other departments (brief): 30 seconds each
- Transitions: 3-5 seconds each

## **TECHNICAL SETUP FOR RECORDING**

### **Browser Settings:**
- Use Chrome or Firefox (best animation support)
- Window size: 1920x1080 (or 16:9 ratio)
- Zoom: 100%
- Clear browser cache before recording
- Disable browser extensions (except if needed)
- Set browser to incognito/private mode for clean address bar

### **System Preparation:**
- Close unnecessary applications
- Disable notifications (Focus Assist on Windows, Do Not Disturb on Mac)
- Hide desktop icons (if showing desktop)
- Clean taskbar/dock
- Full screen browser (F11)

### **Test Data:**
- Verify all demo data is loaded
- Check that all department pages work
- Test chat responses are functioning
- Ensure stats are populated
- Verify animations are smooth

### **Backup Plans:**
- Have screenshots of key screens
- Pre-record complex animations if live is choppy
- Have text version of queries ready to paste
- Keep a running instance in another browser tab

---

## **COMMON ISSUES & SOLUTIONS**

### **Animation too fast/slow:**
- Check reducedMotion setting in accessibility controls
- Adjust browser performance settings
- Record at 60fps for smoother playback

### **Chat not responding:**
- Verify backend API is running
- Check browser console for errors
- Have pre-written responses ready as fallback

### **Hover effects not showing:**
- Move cursor more slowly
- Ensure CSS transitions are enabled
- Check for reduced motion settings

### **Navigation breaks:**
- Use browser back button instead of navigation menu
- Refresh page if stuck
- Have direct URLs ready for each page
