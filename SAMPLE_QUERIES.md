## **TIPS FOR USING THESE QUERIES IN VIDEO:**

### **Best Practices:**
1. **Copy-paste queries** exactly as shown for consistency
2. **Wait 2-3 seconds** after sending before speaking
3. **Highlight key metrics** in the response as you narrate
4. **Use variety** - don't show all queries, pick 1-2 per department
5. **Show natural flow** - let typeahead/suggestions guide some queries

### **Visual Focus Points:**
- Numbers and percentages
- Status indicators (âœ“, âš ï¸, ðŸ”´, ðŸŸ¢)
- Action buttons that appear

"The system even suggests next actions..."
"All decisions fully documented for audit trails..."
